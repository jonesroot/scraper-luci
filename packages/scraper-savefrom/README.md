# Youtube Scraper
Scraper Savefrom

## Installation
```sh
npm i @jonesroot/scraper-savefrom
```

## Usage 
Example download youtube video
```ts
// import module
import { savefrom } from '@jonesroot/scraper-savefrom'

const data = await savefrom('https://youtu.be/iik25wqIuFo')
console.log(data) // JSON
```
